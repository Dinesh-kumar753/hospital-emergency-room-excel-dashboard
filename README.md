# 🏥 Hospital Emergency Room Excel Dashboard

## 📌 Overview

This project is an interactive Excel dashboard designed to analyze hospital emergency room data. It provides insights into patient flow, wait times, admission trends, and departmental distribution to support data-driven decision-making.

---

## 🎯 Objective

* Monitor emergency room performance
* Analyze patient admission trends
* Track average wait time
* Evaluate patient satisfaction
* Understand department-wise patient distribution

---

## 📊 Dashboard Features

* Monthly patient analysis (slicer-based filtering)
* Average wait time tracking
* Patient satisfaction score
* Admission vs Non-admission comparison
* Gender distribution analysis
* Age group-wise patient trends
* Department-wise patient distribution
* On-time vs delayed patient attendance

---

## 📈 Key Insights

* Around **52% patients were not admitted**
* Highest patient count observed in **30–39 age group**
* Majority of patients did not require referral (category: "None")
* Approximately **38% cases experienced delays**, indicating improvement area
